HTML & CSS Cheat Sheet

HTML

Tag Name Description
<!DOCTYPE html> Document Type Declaration The document type declaration is required as the first line of an HTML document. The doctype declaration is an instruction to the browser about what type of document to expect and which version of HTML is being used, in this case it’s HTML5.
<head> Head Element The head element contains metadata (general information about an HTML page that isn't displayed on the page itself), inlcuding title and links to stylesheets.
<title> Title Element The title element contains text that defines the title of an HTML document. The title is displayed in the browser's title bar or tab. This element can only be included inside the document's <head> element.
<body> Body Element The <body> element represents the content of an HTML document. Content inside <body> tags are rendered on the web browsers.
<div> Division Element The <div> element is used as a container that divides an HTML document into sections and is short for “division”.
<h1>, <h2>... <h6> <h1>-<h6> Heading Elements HTML uses six different sized heading elements; the largest size header is <h1> with the heading sizes decreasing and <h6> being the smallest header.
<em> Emphasis Element The emphasis element emphasizes text; browsers will render the text inside an tag, italicized by default.
<ol> Ordered List Element The ordered list element creates a list of items in sequential order; each list item is numbered by default.
<ul> Unordered List Element The unordered list element creates a list of items in no particular order; each list item will have a bullet point by default.
<li> List Item Element The list item element creates list items inside <ul> and <ol>.
<br> Line Break Element The line break element creates a line break in text and is useful where a division of text is required. This element does not require a closing tag.

CSS

Syntax Name Description
p {
}		 Type Selector CSS type selectors are used to match all elements of a given type or tag name. In the syntax provided styles will be applied to all <p> tags.
#id-name {
font-weight: bold;
}		 ID Selector CSS ID selector matches elements based on the contents of their id attribute. The values of id attribute should be unique in the entire DOM. For selecting the element having job-title as the value of the id attribute, a # needs to be prepended.
